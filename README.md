# 📝 Text Editor

A simple desktop **Text Editor** built using **Python** and **Tkinter**. This project demonstrates the basics of GUI development and file handling in Python.

## ✨ Features

- 📄 Create a new file
- 📂 Open existing `.txt` files
- 💾 Save text files
- 🖥️ Simple and user-friendly interface
- 📋 Menu bar for file operations

---

## 🛠️ Built With

- Python
- Tkinter
- File Handling

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/sameerkulkarni216/TextEditor.git
```

### Navigate to the Project Folder

```bash
cd TextEditor
```

### Run the Application

```bash
python TextEditor.py
```
